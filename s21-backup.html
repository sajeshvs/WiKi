2 Enable USB Debugging on Your Phone

Follow these steps exactly:

  1. Go to Settings ‚Üí About Phone
  2. Tap Software Information
  3. Tap Build Number 7 times (you'll see "Developer mode enabled")
  4. Go back to Settings ‚Üí Developer Options
  5. Enable USB Debugging
  6. Enable USB Debugging (Security Settings) if available

5 Backup Photos & Videos (DCIM Folder)

This backs up all your camera photos and videos:

üí° Progress: You'll see each file being copied with its transfer speed. This may take 30-60 minutes depending on size.

üë• Backup WhatsApp Clone Account ‚ñº
8 Find Your Clone App Data Location

Samsung's Secure Folder or App Clone stores data in a different location. First, let's find it:

adb shell ls -la /sdcard/

Look for folders like:

  • /sdcard/Android/data/com.whatsapp.w4b/ (WhatsApp Business clone)
  • /sdcard/secure_storage/ (Secure Folder)
  • /data/media/[user_id]/ (Multi-user profiles)
üí° Common Clone Paths:
# List all WhatsApp-related folders adb shell find /sdcard/ -name "*whatsapp*" -o -name "*WhatsApp*"

Once you find the path, backup the clone:

# Example for typical clone location adb pull /sdcard/WhatsApp_2 C:\Samsung_Backup\WhatsApp_Clone # Or if in Android/data folder adb pull /sdcard/Android/data/com.whatsapp.w4b C:\Samsung_Backup\WhatsApp_Clone

üíª Windows MTP Method (Simple & Visual)

1 Connect Phone to Computer
  1. Connect your Samsung Galaxy S21 Ultra to your computer using a USB cable
  2. On your phone, swipe down the notification panel
  3. Tap on "USB for file transfer"
  4. Select "Transferring files / Android Auto"
üí° Tip: If your computer doesn't recognize the phone, try a different USB cable or port
2 Open File Explorer
  1. Press Windows Key + E to open File Explorer
  2. Look for your phone under "This PC" or "Devices and drives"
  3. Double-click to open your phone
  4. Open "Internal storage" or "Phone"
3 Create Backup Folder on Computer

Create a folder on your computer to store all backups:

  1. Right-click on your Desktop or in a location with enough space
  2. Select New ‚Üí Folder
  3. Name it "Samsung S21 Ultra Backup [Today's Date]"
4 Copy DCIM Folder (Photos & Videos)
  1. In your phone's storage, find the DCIM folder
  2. Right-click on it and select Copy
  3. Go to your backup folder and select Paste
  4. Wait for the transfer to complete (this may take 30-60 minutes)
‚ö†Ô∏è Important: Keep your phone connected and screen on during the transfer
5 Copy Download Folder
  1. Find the Download folder in your phone's storage
  2. Copy and paste it to your backup folder
6 Copy WhatsApp Folder
  1. Find the WhatsApp folder
  2. Copy the entire folder to your backup location
  3. This includes all media, databases, and backups
7 Locate and Copy WhatsApp Clone Data

For Samsung Secure Folder:

  1. In File Explorer, look for folders starting with numbers like "150", "151"
  2. These represent different user profiles/secure folder
  3. Open each one and look for WhatsApp folders
  4. Copy any WhatsApp folders you find to your backup

Alternative locations to check:

  • Look for folders named WhatsApp_2 or WhatsApp Business
  • Check inside Android ‚Üí data for com.whatsapp.w4b
üí° Tip: Use File Explorer's search function (top right) to search for "WhatsApp" to find all related folders
8 Copy Other Important Folders

Copy these additional folders if they contain important data:

  • Documents
  • Music
  • Movies
  • Pictures
  • Recordings (voice recordings)

üÜò Troubleshooting Common Issues

‚ùå "adb: command not found" Error ‚ñº

Solution:

  1. Make sure you extracted the platform-tools ZIP file
  2. Open Command Prompt and navigate to the ADB folder:
    cd C:\adb\platform-tools
  3. Try running the command again with .\adb instead of adb
‚ùå "Device Unauthorized" Error ‚ñº

Solution:

  1. Check your phone's screen for an "Allow USB debugging" prompt
  2. Tap "Allow" and check "Always allow from this computer"
  3. If prompt doesn't appear, revoke USB debugging authorizations:
    • Settings ‚Üí Developer Options ‚Üí Revoke USB debugging authorizations
  4. Disconnect and reconnect your phone
‚ùå Transfer Speed is Very Slow ‚ñº

Solutions:

  • Use a different USB cable (preferably USB 3.0 or higher)
  • Connect to a USB 3.0 port (usually blue colored)
  • Close other programs on your computer
  • Keep your phone screen on and unlocked
  • Disable antivirus temporarily during transfer
