This backs up all your camera photos and videos:

üí° Progress: You'll see each file being copied with its transfer speed. This may take 30-60 minutes depending on size.

8 Find Your Clone App Data Location

Samsung's Secure Folder or App Clone stores data in a different location. First, let's find it:

adb shell ls -la /sdcard/

Look for folders like:

  • /sdcard/Android/data/com.whatsapp.w4b/ (WhatsApp Business clone)
  • /sdcard/secure_storage/ (Secure Folder)
  • /data/media/[user_id]/ (Multi-user profiles)
üí° Common Clone Paths:
# List all WhatsApp-related folders adb shell find /sdcard/ -name "*whatsapp*" -o -name "*WhatsApp*"

Once you find the path, backup the clone:

# Example for typical clone location adb pull /sdcard/WhatsApp_2 C:\Samsung_Backup\WhatsApp_Clone # Or if in Android/data folder adb pull /sdcard/Android/data/com.whatsapp.w4b C:\Samsung_Backup\WhatsApp_Clone

üÜò Troubleshooting Common Issues

‚ùå "adb: command not found" Error ‚ñº

Solution:

  1. Make sure you extracted the platform-tools ZIP file
  2. Open Command Prompt and navigate to the ADB folder:
    cd C:\adb\platform-tools
  3. Try running the command again with .\adb instead of adb
‚ùå "Device Unauthorized" Error ‚ñº

Solution:

  1. Check your phone's screen for an "Allow USB debugging" prompt
  2. Tap "Allow" and check "Always allow from this computer"
  3. If prompt doesn't appear, revoke USB debugging authorizations:
    • Settings ‚Üí Developer Options ‚Üí Revoke USB debugging authorizations
  4. Disconnect and reconnect your phone
‚ùå Transfer Speed is Very Slow ‚ñº

Solutions:

  • Use a different USB cable (preferably USB 3.0 or higher)
  • Connect to a USB 3.0 port (usually blue colored)
  • Close other programs on your computer
  • Keep your phone screen on and unlocked
  • Disable antivirus temporarily during transfer

# List all storage locations adb shell ls -la /storage/ # Search entire phone for WhatsApp folders adb shell find /sdcard/ -type d -iname "*whatsapp*" 2>/dev/null # Check Android/data for cloned apps adb shell ls -la /sdcard/Android/data/ | grep -i whatsapp
